# Contributing

This repository is the **Crateria org landing page**. It does not contain product
source code.

- Product changes → open PRs on [trance](https://github.com/crateria/trance),
  [trance-plugins](https://github.com/crateria/trance-plugins),
  [morphball](https://github.com/crateria/morphball), or
  [packages](https://github.com/crateria/packages).
- Brand / icons → [crateria/brand](https://github.com/crateria/brand).
- Keep this README accurate when products are added or renamed.

See also: https://github.com/crateria/.github/blob/main/CONTRIBUTING.md
