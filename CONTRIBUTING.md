# Contributing

This repository is the Crateria organization map. It does not contain product
source code.

| Change type | Where to contribute |
|-------------|---------------------|
| Screensaver / daemon / CLI | [crateria/trance](https://github.com/crateria/trance) |
| Effects / plugins | [crateria/trance-plugins](https://github.com/crateria/trance-plugins) |
| Archive tool | [crateria/morphball](https://github.com/crateria/morphball) |
| Packages / indexes | [crateria/packages](https://github.com/crateria/packages) |
| Icons / brand | [crateria/brand](https://github.com/crateria/brand) |
| Org profile text | [crateria/.github](https://github.com/crateria/.github) (`profile/README.md`) |

Keep this README accurate when products are added or renamed.

Organization guide: https://github.com/crateria/.github/blob/main/CONTRIBUTING.md
